## 화폐
Smart Contract를 위해 비트코인의 원리를 이해해야 한다. <br />

서울에서 페소, 달러 등의 외화는 거래 수단이 되지 못하지만, 원화는 모두가 인정하며 사용한다. **즉, 돈은 그 자체로 가치 있는 것이 아닌, 사용 되는 장소의 공동체가 그것을 '신뢰'하고 '보증'**하기 때문에 ***가치 있는 것이다.***
